Perfil del Inversor


El sistema es una herramienta interactiva que evalúa el 
perfil de riesgo de inversión de un usuario. 
Mediante una serie de preguntas cuidadosamente diseñadas, 
el sistema mide el horizonte temporal y la tolerancia al 
riesgo del usuario. Con esta información, genera un perfil 
personalizado que ayuda a identificar cómo distribuir mejor 
las inversiones en función de las características personales 
y los objetivos financieros.

* Basado en el cuestionario de Charles Schwab
